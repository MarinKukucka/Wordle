# Wordle
> A simple, fun Wordle clone built with React. Guess the 5-letter word in 6 tries.

## Demo
Try it live: <https://wordle-alpha-vert.vercel.app/>

---

## Features
- Classic Wordle mechanics: 6 guesses, feedback by letter (correct / present / absent)
- Dictionary with over 3500 english words
- Explanation of solution on end screen
- Keyboard + mobile-friendly controls
- Unlimited play

## Tech Stack
- Frontend: `TypeScript + React 19`
- State management: `plain useState`
- Styling: `CSS Modules`

## Installation (development)
```bash
# clone
git clone https://github.com/MarinKukucka/Wordle.git
cd Wordle

# install
npm install

# start dev server
npm start
